# Hello World!

This is a collection of various hello world programms in different languages, used for teaching.
PRs welcome!

## C

- compile:

```
gcc HelloWorld.c -o HelloWorld
```

- run:

```
./HelloWorld
```

## Java

- compile:

```
javac HelloWorld.java
```

- run:

```
java HelloWorld
```

## JavaScript

- run:

```
node HelloWorld.js
```

## Kotlin

- compile:

```
kotlinc HelloWorld.kt
```

- run:

```
java HelloWorldKt
```

## Python

- run:

```
python HelloWorld.py
```
